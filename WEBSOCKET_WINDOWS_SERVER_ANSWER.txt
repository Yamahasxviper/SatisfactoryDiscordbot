================================================================================
COMMON ISSUES & SOLUTIONS
================================================================================

Issue: "WebSocket connection failed"
  Solution: Check Windows Firewall, allow FactoryServer.exe outbound

Issue: "Bot presence not updating"
  Solution: Enable UseGatewayForPresence=true in config
            Enable "Presence Intent" in Discord Developer Portal

Issue: "Disconnects frequently"
  Solution: Check network stability, verify internet connection

Issue: "Won't work on Windows 7"
  Solution: Upgrade to Windows 8.1+ or Server 2012 R2+ (required)
            Or use Linux dedicated server instead
